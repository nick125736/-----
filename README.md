`ytcomment.py` 為取得YT留言的簡易python

且也可以取得留言的回覆

並從陣列轉存



`ytinfor.py` 為取得影片詳細資訊

``` python
part="snippet,contentDetails,statistics"
```

snippet為詳細資訊\contentDetails影片長度等\statistics影片觀看數詳細資訊
